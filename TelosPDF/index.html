TelosPDF

The open-source PDF workstation. Everything paid PDF software charges for — editing, signing, redaction, OCR — free, offline, and MIT-licensed, in a fast modern workbench.

Download View source on GitHub
Everything a PDF workstation should do

View & navigate

Continuous scroll, tabs, thumbnails, bookmarks, search with highlights, trackpad-smooth zoom — pages render in a Rust core, never in the webview.

Edit PDF

Click any text or object on the page to edit or delete it; add new text directly onto the page.

Organize pages

Reorder by drag, rotate, delete, extract, insert blank pages — with live thumbnails.

Fill & Sign

Fill form fields, draw your signature once and reuse it, stamp APPROVED/DRAFT and more. Move placements freely and undo until you hit Save.

Markup

Freehand ink, shapes, arrows, and text boxes, baked permanently into the PDF.

Comments

Sticky-note comments with colours, tracked in a side panel — interoperable with other PDF apps.

True redaction

Content is permanently removed from the file — not drawn over. Verified by adversarial extraction tests.

Scan & OCR

Bundled Tesseract makes scans searchable offline. Optional: Baidu's Unlimited-OCR 3B model for state-of-the-art accuracy on scans, tables, and CJK text.

Compare files

Side-by-side Original / New / Differences views with text-level diffing.

Convert

Create PDFs from Office documents and export PDFs back to Word, Excel, and PowerPoint.

Protect, unlock & compress

AES-256 password protection, permission controls, password removal, and structural compression via qpdf.

Print

A real print dialog: printer choice, page ranges, odd/even sets, copies, and a live queue.

Why TelosPDF
  • ✓100% free and MIT-licensed — for individuals and enterprises, no feature gates
  • ✓Fully offline: no account, no telemetry, documents never leave your machine
  • ✓PDF JavaScript is never executed — malicious PDFs stay inert
  • ✓Light, Dark, and System themes — including a pure-black OLED mode
  • ✓Rust core on Chrome's PDFium engine; permissive licensing enforced in CI
  • ✓macOS (Apple Silicon) and Windows today — Linux and Android APK on the roadmap
